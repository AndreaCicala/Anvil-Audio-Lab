What it does

Four tools. One app. Zero uploads.

Each tool reads your audio locally, runs analysis, shows you what's going on. Your files stay on your drive.

◆

Mix analysis

LUFS + true peak, spectral balance against genre targets, per-section breakdown, reference overlay, frequency cutoff detection, stereo image, dynamics — everything you'd want in a mix report, in one page.

images/mix-analysis.png
◈

Stem masking

Drop multiple stems, get a masking matrix heatmap, per-band conflict detection, and EQ cut recommendations scoped to each source track — not the bus. With version diffing so you can see if your fix worked.

images/stems.png
◎

Offline mastering

Gentle limiter, tilt shelf, M/S bass check, sides-air check, 6-band EQ, reference match, genre match. Every move is numeric and inspectable. A/B against your source with a click, export 24-bit WAV with metadata.

images/master.png
✓

Release check

Drop your finished master and verify it meets delivery specs for Spotify, YouTube, or Bandcamp. Pass / warn / fail on LUFS, true peak, LRA, crest, correlation, mono-bass, and frequency cutoff. No surprises at the distributor.

images/release.png
Why

The cloud was the wrong shape.

I built Anvil Audio Lab because I was tired of uploading my mixes to cloud services every time I wanted a second opinion. I produce post-rock, prog, and metal — the kind of music where a single track is a 5-minute 96 kHz WAV that weighs 250 MB. Every analysis meant waiting for an upload, waiting for a server, and trusting that the service didn't keep a copy.

The tools I wanted all existed as plugins inside a DAW, not as something I could point at an exported WAV. And the web-based alternatives were locked behind monthly subscriptions for features I'd use a few times a month.

So I built my own. It runs on my laptop, reads my files directly, and shows me what I need to see — nothing more. If it's useful to you too, that's a bonus. Take it, use it, mix better.

Install

Three steps.

No installer. No setup wizard. No Python or ffmpeg to install separately.

01
Download
Grab the zip (307 MB) from the button above or the releases page.
02
Unzip
Extract anywhere you like — Desktop, Documents, Program Files. Inside is a folder called AnvilAudioLab.
03
Run
Double-click AnvilAudioLab.exe. A console opens, then your browser at localhost:5000.
