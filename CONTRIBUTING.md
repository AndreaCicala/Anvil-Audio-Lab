# Contributing to Anvil Audio Lab

Thanks for the interest, but **this is a personal project and I'm not
accepting contributions at this time.** That includes:

- Pull requests
- Feature requests
- Bug reports
- Code reviews

The repo is public so people can read the code, learn from it, and fork
it for their own use. I may open it up to contributions in the future, but
not yet.

## What you *can* do

- **Fork the repo** and hack on your own copy freely.
- **Use the tool** — download a release and run it on your own mixes.
- **Check for updates** via the in-app version chip.

## If you think you've found a security issue

Email me rather than opening a public issue (contact info on my GitHub
profile). Standard responsible-disclosure appreciated.
